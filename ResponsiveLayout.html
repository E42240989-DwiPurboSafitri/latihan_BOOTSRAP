Responsive Layout

Apa itu Responsive Layout?

Responsive layout adalah layout yang otomatis menyesuaikan tampilan sesuai ukuran layar. Bootstrap menggunakan grid system dengan 12 kolom. Contoh: pada layar besar, konten bisa 3 kolom sejajar, tapi di layar kecil akan otomatis menumpuk jadi 1 kolom.
